Installing on a Mac

Open the .dmg and drag Game DJ into your Applications folder. That's it. The app is signed and notarised by Apple, so it opens without any security warnings.

What you need

macOS 11 (Big Sur) or later on an Apple Silicon Mac — M1, M2, M3 or M4. There's no Intel build right now, so a pre-2020 Mac won't run it. If that's you, say so by email — enough demand and it's an easy build to add.

Updates

Game DJ updates itself. New versions download quietly in the background and install when you restart the app, so you're never the one chasing a download link before a game.

The trial

The 14 days start the first time you open the app, and nothing is locked while they run. When they're up, subscribe from inside the app — the Buy button opens a secure Paddle checkout and the app activates itself the moment payment clears.
