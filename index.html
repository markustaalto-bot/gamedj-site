The soundboard

Every sound one key away

Open the app and 20 stadium sounds are already there. Add your own MP3s and WAVs alongside them, group everything into categories, and bind each pad to a number key. Trim the dead air with cue points so the horn hits on the first frame — not half a second later.

  • 20 built-in sounds: horns, buzzers, whistles, fanfares, crowds
  • Cue points for start and end, plus fades
  • A number key per sound — 1–9
  • Record an ad straight from your mic
The timeline

Script the night in advance

Lay out the warm-up, the line-ups and the sponsor reads in the order they happen. Hit one key to move to the next clip and Game DJ crossfades it in. The clock above each clip tells you when it will actually play.

  • Drag sounds in from any category
  • Smooth crossfades between clips
  • Predicted wall-clock times that follow the real start
  • Live playhead so you always know where you are
The announcer

Goal announcements that read themselves

Import both rosters from an Excel file, type the scorer's number, and Game DJ speaks the whole announcement — name, assist and the new score — in Finnish or English. Got the scorer wrong? Fix the event and it announces the correction.

  • Floorball, ice hockey and football, each with its own wording
  • Rosters straight from Excel — no retyping
  • Every sentence is a template you can rewrite
  • Export the match events back to Excel

Questions, answered

Can I try it before paying?
Yes. Every download includes a 14-day trial with all features unlocked. No card, no account — just install it and go. When the trial ends you subscribe from inside the app.
Which sports does it cover?
Floorball, ice hockey and football each have their own event types and announcement wording. The soundboard and timeline are sport-agnostic, so plenty of people use Game DJ for events that aren't in that list at all.
Does it need an internet connection?
Not for the part that matters. Local sounds, cue points, hotkeys and the timeline all run entirely on your machine, so a dead arena Wi-Fi can't take your goal horn down. Spotify and generating new spoken announcements do need a connection — announcements you generated earlier are saved as normal audio files and keep working offline.
Do I need Spotify Premium?
Only if you want the Spotify feature — everything else works without it. There's nothing to set up: Game DJ drives the Spotify app already open on your machine, with no login and no developer keys. Spotify itself only allows that remote control on Premium accounts. On a Mac you also get Apple Music the same way, if you have a subscription.
Does the subscription cover music licensing?
No — and no software can. Public performance of music at an event needs the appropriate licences (in Finland, Teosto and Gramex). That stays the organiser's responsibility. Game DJ plays the audio; it doesn't grant rights to it.
What do I need to run it?
A Mac with Apple Silicon (M1 or newer) running macOS 11 or later. A Windows version is in the works. See the download page for the current state.
How do I cancel?
Any time, from the receipt email Paddle sent you or by emailing gamedj@waveric.fi. You keep access until the end of the period you already paid for. There's also a 14-day money-back guarantee on your first payment.
